Obavještavam vas o novom značajnom poboljšanju koje omogućava izuzetno brz rad s narudžbama putem tipkovnih prečaca:

⌨️ Potpuni tipkovni workflow za narudžbe
  • F1 → Idite na kreiranje narudžbe (iz bilo kojeg dijela aplikacije)
  • Auto-fokus → Pretraga ljekarne se automatski aktivira
  • ↑↓ + Enter → Navigirajte i odaberite ljekarnu
  • D ili A → Dodaj proizvod
  • ↑↓ + Enter → Navigirajte i odaberite proizvod
  • Auto-fokus količine → Polje količine se automatski označava
  • K ili C → Kreiraj narudžbu
  • F2 → Povratak na popis narudžbi
🎯 Pametne funkcionalnosti
  • Prečaci ne ometaju tipkanje - automatski se prepoznaje kada tipkate u polja
  • Vizualno označavanje odabranih stavki (plavo pozadinsko osvjetljenje)
  • Escape tipka za zatvaranje dijaloga i pretraga
  • Zaštićeno od slučajnog stvaranja duplikatnih narudžbi
🔧 Novo: Pametno uređivanje završenih narudžbi

⚠️ VAŽNO: Promjena u radu sa završenim narudžbama

Ranije niste mogli uređivati završene narudžbe. Sada možete, ali drugačije:

🔄 Kako sada funkcionira:
  1. Kliknete "Edit" na završenoj narudžbi (gumb je sada vidljiv)
  2. Aplikacija pita potvrdu: "Narudžba je završena. Želite li je staviti na čekanje za uređivanje?"
  3. Ako kliknete "Da": Status se mijenja na "pending" i postojeći E-račun se briše
  4. Sada možete uređivati narudžbu kao i obično
  5. Nakon uređivanja: Označite narudžbu kao završenu i novi E-račun će se generirati
✅ Prednosti novog sustava:
  • Nema više grešaka "duplicate key constraint" kod uređivanja
  • Sigurno upravljanje E-računima (stari se briše, novi se stvara)
  • Jasno označavanje što se događa s potvrdom
  • Stiliziran dijalog umjesto osnovnog browser upozorenja
💡 Napomena: Ova promjena rješava problem s duplikatnim računima koji se dogodio ranije i čini uređivanje sigurnijim.
Praktični primjer brzog rada:
F1 → počnite tipkati "Dom" → ↓↓ Enter → D → tipkajte "parace" → ↓ Enter → upišite "5" → C
Cijela narudžba kreirana u 10-15 sekundi bez korištenja miša!

⚡ Ovo omogućava izuzetno brz rad: Iskusni korisnici mogu kreirati narudžbu kompletno pomoću tipkovnice u manje od 20 sekundi!
